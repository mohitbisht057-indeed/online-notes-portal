# 📚 Notes Portal Management System

A web-based Notes Portal developed using PHP and MySQL that allows students to securely upload and access study materials online.

## 🚀 Features

- User Registration
- User Login
- Session Management
- Upload PDF Notes
- View Uploaded Notes
- Open PDF Files
- Logout Functionality
- Responsive User Interface

## 🛠️ Technologies Used

- HTML
- CSS
- PHP
- MySQL
- XAMPP
- InfinityFree (Hosting)

## 🗄️ Database Tables

### users

| Field | Type |
|---------|------|
| id | INT |
| name | VARCHAR |
| email | VARCHAR |
| password | VARCHAR |

### notes

| Field | Type |
|---------|------|
| id | INT |
| filename | VARCHAR |
| Name | VARCHAR |
| Email | VARCHAR |
